# Simple Second Brain App

Get thoughts of your head and into a digital space. Keep notes and track necesseties without worry. 

### This app is still in progress!

## Current features:

**Notes**
- Create simple notes
- Sort notes into folders
- Assign categories to notes
- Created notes in inbox for sorting
- Hardcoded folder structure
- Add, edit and delete subfolders

**Income Tracker**
- Track income made per week
- Add single income entry via form or multiple entries in text area 
- Set weekly target goal
- Income recorded in Weekly, Monthly, and Yearly views
- Views each have bar charts and data views to quickly and easily comprehend income

**Expenses Tracker**
- Track one-time and recurring expenses
- Set name, amount, due date, and payment state for one-time expenses
- Set name, amount, initial due date, payment state, and recurrence frequency and amount
- Expenses recorded in Monthly and Overall views
